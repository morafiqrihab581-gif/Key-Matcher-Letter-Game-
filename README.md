# 🎮 Key Matcher (Letter Game)

## 🧠 Description
Key Matcher is a simple JavaScript game where a random letter is displayed on the screen.  
The player must press the matching key on the keyboard.

---

## 🎯 Features
- Random letter generation (A–Z)
- Keyboard event detection
- Visual feedback (correct / incorrect colors)
- Enter key generates a new letter
- Smooth UI transitions

---

## 🎮 How to Play
1. A random letter appears on screen
2. Press the matching key on your keyboard
3. If correct → next letter appears
4. Press Enter → skip and get a new letter

---

- JavaScript (DOM + Keyboard Events)
